SUGGESTED EXAMPLES TO DEMONSTRATE:
----------------------------------

Example 1: Squaring array elements

   Vectorized: result = arr ** 2

Example 2: Element-wise arithmetic
   Loop: for i in range(len(arr)): result[i] = arr1[i] + arr2[i]
   Vectorized: result = arr1 + arr2

Example 3: Conditional counting
   Loop: count = 0; for x in arr: if x > threshold: count += 1
   Vectorized: count = np.sum(arr > threshold)
