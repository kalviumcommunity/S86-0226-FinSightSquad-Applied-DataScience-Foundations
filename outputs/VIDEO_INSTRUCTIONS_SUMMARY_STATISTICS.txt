TECHNICAL REQUIREMENTS
-----------------------

Screen Recording Setup:
- Full screen of your terminal/IDE/Jupyter notebook
- Ensure all text is clearly readable
- No camera required (screen-only recording)
- Good audio quality for narration

Code to Demonstrate:
```python
import pandas as pd

# Load data
df = pd.read_csv('sample_transactions.csv')

# Part 1: Select column
amount = df['Amount']
print(amount.head())

# Part 2: Compute statistics
print(f"Mean: ${amount.mean():.2f}")
print(f"Median: ${amount.median():.2f}")
print(f"Min: ${amount.min():.2f}")
print(f"Max: ${amount.max():.2f}")
print(f"Std: ${amount.std():.2f}")
print(amount.describe())

# Part 3: Interpret (show comparison)
print(f"Mean vs Median: ${amount.mean():.2f} vs ${amount.median():.2f}")

# Part 4: Compare categories
print(df.groupby('Category')['Amount'].mean())
```

Output Requirements:
- All statistics should be clearly visible
- Numbers should be formatted properly (e.g., $123.45)
- Output should not scroll off screen too quickly
- Pause briefly on important results
