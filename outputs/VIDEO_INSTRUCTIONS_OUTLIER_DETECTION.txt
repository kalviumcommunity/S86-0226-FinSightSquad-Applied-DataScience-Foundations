Code to show:
```python
q1 = df['Amount'].quantile(0.25)
q3 = df['Amount'].quantile(0.75)
iqr = q3 - q1
lower_bound = q1 - 1.5 * iqr
upper_bound = q3 + 1.5 * iqr

outliers = df[(df['Amount'] < lower_bound) | (df['Amount'] > upper_bound)]
print(f"Lower bound: ${lower_bound:.2f}")
print(f"Upper bound: ${upper_bound:.2f}")
print(f"Flagged outliers: {len(outliers)}")
print(outliers[['Date', 'Category', 'Amount']])
```

Recommended Code Flow:
```python
import pandas as pd
import matplotlib.pyplot as plt

# Load data
df = pd.read_csv('sample_transactions.csv')

# Step 1: Describe the column
print(df['Amount'].describe())

# Step 2: Boxplot
plt.boxplot(df['Amount'])
plt.title('Amount Distribution')
plt.show()

# Step 3: IQR Rule
q1 = df['Amount'].quantile(0.25)
q3 = df['Amount'].quantile(0.75)
iqr = q3 - q1
lower = q1 - 1.5 * iqr
upper = q3 + 1.5 * iqr
outliers = df[(df['Amount'] < lower) | (df['Amount'] > upper)]
print(f"Flagged outliers:\n{outliers[['Date', 'Category', 'Amount']]}")

# Step 4: Interpret
print("These values stand out because they exceed the expected range.")
print("Investigation is needed before any action is taken.")
```
