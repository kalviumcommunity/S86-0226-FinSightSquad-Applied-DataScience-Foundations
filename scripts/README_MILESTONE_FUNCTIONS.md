Functions Milestone — `scripts/functions_milestone.py`

Usage

- Run the functions milestone examples:

```bash
python scripts/functions_milestone.py
```

What it contains

- Examples of defining functions with `def`.
- Positional and keyword arguments, default parameters.
- Return values and printing vs returning.
- A small recursive example (`factorial`).
- A demonstration of local vs global scope (`GLOBAL_COUNT`).

Video and submission

- Record a ~2-minute screen-capture demonstrating: defining a function, calling it with arguments, and explaining execution flow and scope.
- Save the recording and place a shareable link in `outputs/video_link.txt` as instructed by your course.
