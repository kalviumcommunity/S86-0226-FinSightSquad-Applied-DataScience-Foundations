# Using for and while Loops for Iterative Data Processing Milestone

## Purpose

This milestone focuses on using `for` and `while` loops for iterative data processing in Python.

Loops help you:

- avoid repetitive code
- process collections efficiently
- control repeated execution with clear logic
- write scalable and maintainable scripts

---

## Run the Script

```bash
python scripts/for_while_loops_milestone.py
```

---

## What the Script Demonstrates

### 1) Using `for` Loops for Iteration

- Iterating over `range()` for known sequences
- Iterating over a list of values
- Using loop variables meaningfully
- Observing execution order through printed output

### 2) Using `while` Loops for Condition-Based Repetition

- Repeating while a condition is true
- Updating loop variables correctly
- Stopping when the condition changes
- Understanding when `while` is the right tool

### 3) Controlling Loop Flow

- Using `continue` to skip invalid iterations
- Using `break` to exit early when needed
- Keeping loop logic readable and predictable

### 4) Avoiding Infinite Loops

- Identifying common infinite-loop causes
- Ensuring loop conditions are updated
- Applying a safety limit (`max_attempts`) pattern
- Testing small examples first

---

## Expected Learning Outcomes

After completing this milestone, you should be able to:

1. Write `for` loops to process ranges and lists
2. Write `while` loops using condition-based repetition
3. Use `break` and `continue` appropriately
4. Recognize and fix infinite loop risks
5. Apply loop logic confidently in simple data workflows

---

## Common Beginner Pitfalls

1. **Repeating code manually** instead of using loops
2. **Forgetting to update loop variables** in `while` loops
3. **Using the wrong loop type** for the scenario
4. **Confusing loop flow** when mixing too many conditions
