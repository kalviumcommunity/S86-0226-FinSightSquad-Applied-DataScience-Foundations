Numeric + String Data Types Milestone

Purpose

- Practice Python core data types before dataset work.
- Understand differences between numbers (`int`, `float`) and text (`str`).
- Build safe habits for conversions and type checks.

Run

```bash
python scripts/numeric_string_milestone.py
```

What the script demonstrates

- Numeric variables and arithmetic operations.
- String variables, concatenation, indexing, and formatting.
- Type mismatch behavior (captured `TypeError` example).
- Explicit conversions with `str()` and `int()`.
- Type inspection with `type()` and `isinstance()`.

Expected learning outcomes

- Differentiate numeric vs string data types.
- Predict arithmetic and concatenation behavior.
- Identify and fix type-related issues early.
- Write cleaner, more reliable beginner Python code.
