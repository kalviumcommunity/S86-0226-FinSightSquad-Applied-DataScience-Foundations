# Collections Milestone — Lists, Tuples, Dictionaries

This script demonstrates basic usage of Python collections: lists, tuples, and dictionaries.

What to run

Run the example script to see output demonstrating creation, access, modification (where allowed), and iteration:

```bash
python scripts/collections_milestone.py
```

Video guidance (2 minutes)

- Show running the script and explain each section on-screen:
  - The list example: creation, indexing, append, modify, pop, iterate.
  - The tuple example: creation, indexing, show that modification raises an error.
  - The dictionary example: creation, key access, modify/add, iterate keys/values.
- Briefly explain when to choose list vs tuple vs dict.
